PENYELESAIAN :
Praktikum Sistem Informasi Geografis Berbasis Web ini sudah menyelesaikan praktikum dari minggu 1 hingga minggu 7. SIG berbasis Web ini memiliki peran penting untuk penyelesaian masalah berbasis SIG yang dimana berguna untuk visualisasi hasil penyelesaian masalah daapat berbentuk dashboard yang sudah dipublikasi, maupun dalam bentuk bootstrap. Praktikum Minggu 1-2, menjelaskan kategori penyelesaian untuk membuat tampilan website sesuai yang diperintah, dan disesuaikan dengan rules sesuai tugas yang ada di modul. Adapun itu dengan tampilan gambar kucing dan pembuatan form dengan jenis kucing dan asal jenis kucing tersebut. Pada tugas nomor 2, diperintahkan untuk pembuatan website yang dimana menampilkan informasi daerah Prancis dengan tampilan navigasi,home, gambar dari wilayah Prancis, penjelasan wilayah Prancis yang menarik, dan lain-lain. Lalu pada soal ketiga, diminta membuat profil CV kita melalui website Wix.com yang dimana membuat portofolio kita selama berkegiatan perkuliahan di Teknik Geodesi UGM ini. Praktikum Minggu 3-4, menjelaskan kategori pembuatan input angka dan pembuatan angka fibbonacci, faktorial, dan-lain-lain. Adapun maksud dan tujuan pembelajaran ini ialah untuk mengetahui penggunaan html yang dimana menjadi kerangka awal untuk pembuatan website. Rules sederhana pada tugas di minggu 3 ini, menjadi bekal untuk pembelajaran tugas di minggu 4. Praktikum Minggu 5, menjelaskan kategori pembuatan cv menggunakan bootstrap dan dilakukan publikasi hasil public cv, dengan memberikan link hasil ke tugas form. CV berisi pengalaman selama berkuliah di Teknik Geodesi UGM, yang menggunakan repository github.com. Praktikum Minggu 6, menjelaskan kategori pembuatan Peta Leaflet.js yang dimana membuat peta OSM dengan bantuan CSS,HTML,Java Script. Pembuatan tugas ini meliputi atribut marker, dan heading website. Lalu hasil pembuatan peta rute tersebut, dimasukkan hasilnya dan dipublikasi pada github.com untuk dapat dilakukannya publikasi. Praktikum Minggu 7, menjelaskan kategori pembuatan Peta Leaflet.js yang dimana hampir sama dengan pembuatan tugas Praktikum Minggu 6, akan tetapi ada pembaharuan kategori yaitu penambahan beberapa atribut berupa marker, kolom pencarian dan navgiasi lokasi user saat ini, dan layer-layer peta yang dapat diakses user, dan layer nya pun ditambahkan dari file shp yang diconvert ke file format geojson untuk diambilnya data koordinat wilayah Daerah Istimewa Yogyakarta untuk pembuatan klasifikasi di tiap kota dan kabupatennya di wilayah DIY.
